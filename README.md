## Installation

We advise you to install all devdependencies locally (linters, etc):

`npm install`

Don't forget to create `.env` (from `.env.example`).

## Run

To run use:

`npm run start`

To run the dev version use:

`npm run dev`

To run the eslint use:

`npm run test`
